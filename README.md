# Build Flow
- Trainモードで学習を行う。
- Testモードに遷移すると学習が始まる。
- 推論結果に反映される。　　

# Control 
- キーボード "1"でTestモード  
- キーボード "2"でTrainモード  
- キーボード "a"でMouseの位置に対する推論  
- キーボード "s"でmyfigureで選択した図形が描かれる。
